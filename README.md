# claude-agents

Multi-agent Claude Code setup with skills for task planning,
architecture, coding, review, and testing.

## Agents
- task-planner — orchestrates all work
- architecture-agent — system design decisions
- coding-agent — feature implementation
- code-assistant — inline help and edits
- code-reviewer — quality and security review
- testing-expert — test writing and coverage

## Usage
Run `claude` inside any project, then invoke with /task-planner, etc.
